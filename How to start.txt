SmartTag — AI Toll Gate Monitor v3
===================================

QUICK START:

python -m venv venv

venv\Scripts\activate

python -m pip install --upgrade pip

1. Install dependencies:
   pip install -r requirements.txt

2. Start the server:
   python run.py

   OR from backend directory:
   uvicorn app.main:app --reload --port 8000

3. Open browser: http://localhost:8000

LOGIN CREDENTIALS:
  superadmin : admin / admin@123
  operator   : operator1 / op@pass123
  analyst    : analyst / analysis@456
  viewer     : viewer / view@789

ROLES & ACCESS:
  superadmin - Full access, can manage all users
  admin      - Can manage users, vehicles, fastag, all reports
  analyst    - Can view all data, resolve fraud alerts
  operator   - Can process vehicles, manage vehicles/fastag, view transactions
  viewer     - Read-only access to dashboard and transactions
